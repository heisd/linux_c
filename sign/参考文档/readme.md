# 信号
## 什么是信号
信号是信息的载体，linux/unix
现在信号还是两个进程主要的通信手段之一

## 信号的特点
### 简单
### 不能携带大量信息
### 满足某个特点条件才会产生

## 信号的机制
进程在收到信号时，不论程序运行到程序的什么位置，都要暂停运行，去处理信号，处理完毕之后再继续执行
